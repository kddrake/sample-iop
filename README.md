#**sample-iop**
A sample and test of the UX Intern Onboarding Project

##**How to Use**
1. In the root directory of this repository run `gulp`
2. Load `localhost:1820/src` in your preferred browser

1. angular
  1. angular-resource
  2. angular-ui-router
  
##**DevDependencies**

  1. gulp-connect
  2. gulp-sass
  3. gulp-help
  4. gulp-jshint
